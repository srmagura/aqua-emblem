Controls

Arrow keys: move cursor / switch tabs
F: "A button" - select / choose
D: "B button" - back
S: more info (units, skills, items)
E: end turn
V: skip text / toggle 3x speed

How to play

View the Game Guide

Use F to select your units (blue) and move them around the map. Defeat the enemy army to beat the chapter. Click the button above to view the full Game Guide.

  • You lose if ANY of your units die!
  • Use Arrow's First Aid skill to heal your units! Units can even use healing skills on themselves. You'll get more healing skills as the game goes on.

Supported browsers

The latest desktop versions of Chrome, Firefox, Safari, and Microsoft Edge are supported. Mobile devices are not officially supported.
