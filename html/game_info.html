Controls

Arrow keys: move cursor / switch tabs
F: select / choose
D: back
S: more info (units, skills)
E: end turn

Release notes for demo-v0.001

  • Units gain experience in each battle. Once a unit builds up 100 exp, they level up.
  • Trade items between units by choosing the "Trade" option from the action menu.
  • Skills system. Choose "Skill" from the action menu to use a skill. There is only one skill right now (Defend), which everyone has. More skills will be added in the future. A "Skills" tab has also been added to the unit info window (which appears when you press S while the cursor is over a unit.)
  • Using a skill takes a certain amount of MP. A unit regains 1 MP for every regular attack it lands.
  • Status system. Units that have a special status are marked with an orange circle. Move the cursor to a unit to view its list of statuses. There is currently only one status, Defend, which signifies that a unit has used the Defend skill.
  • Base stats and growth rates have been rebalanced. I'll continue to rebalance things as new skills, units, and chapters are added.
  • Kenji has received cosmetic plastic surgery, becuz he was ugli before.
This version has been tested in Firefox and Chrome. Please tell me if you find a bug.
