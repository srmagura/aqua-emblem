Controls

Arrow keys: move cursor / switch tabs
F: "A button" - select / choose
D: "B button" - back
S: more info (units, skills, items)
E: end turn
V: skip text / toggle 3x speed

How to play

View the Game Guide

Aqua Emblem is a lot like Fire Emblem. Use F to select your units (blue) and move them around the map. Defeat the enemy army to beat the chapter. Click the button above to view the full Game Guide.

  • You lose if ANY of your units die!
  • Use Arrow's First Aid skill to heal your units! Units can even use healing skills on themselves unlike in Fire Emblem. You'll get more healing skills as the game goes on.

Supported browsers

Aqua Emblem is tested in Chrome, Firefox, and Internet Explorer 11. Mobile devices are not officially supported. The game probably works in Safari, but I don't test in Safari.

About

Aqua Emblem was developed by me: Sam Magura, srmagura AT ncsu.edu. I originally planned for Aqua Emblem to be a 10 chapter game, but I had only gotten to Chapter 3 when school started. So I decided to just release what I had.

Aqua Emblem is open source. Please email me if you have feedback, or are interested in continuing the game's development. The code is online at the GitHub repository. Special thanks to CoffeeScript. Aqua Emblem is copyright 2014 Sam Magura. Released under the GNU General Public License.
